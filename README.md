# Live-API-Data-Integration-Power-BI-Dashboards
A real-time analytics project that integrates live public API data with Power BI dashboards, enabling automated refresh and continuous monitoring of global socio-economic and health indicators.

Objectives
Fetch and process live API data using Python
Enable automatic dashboard refresh in Power BI
Analyze global development, health, and economic trends

Data Source
Public API providing updated global indicators (health, education, economy, environment)

Tools & Technologies
Python (API requests, data processing)
Power BI
Data modeling and visualization

Dashboard Analysis
GDP per capita and growth trends
Health expenditure (% of GDP)
Internet and mobile penetration
Unemployment and poverty trends
Relationships between:
Internet access and immunization rates
Internet access and unemployment
Health spending and life expectancy

Impact
Supports:
Policy analysis
Development planning
